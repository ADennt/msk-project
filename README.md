# МСК — Дорожные знаки (GitHub Pages)

Это интернет-магазин дорожных знаков, работающий полностью в браузере. Данные хранятся в localStorage, что позволяет редактировать товары, оформлять заказы и управлять ими без сервера.

## 🚀 Как запустить проект локально

1. Скачайте все файлы в одну папку.
2. Откройте `index.html` в браузере (двойной клик).
3. Готово! Сайт работает.

## 🌐 Как разместить на GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите все файлы в корневую папку репозитория.
3. Зайдите в настройки репозитория → Pages.
4. В разделе «Branch» выберите `main`, а папку — `/ (root)`.
5. Нажмите «Save».
6. Через минуту сайт будет доступен по адресу:
   `https://ВАШ_ЛОГИН.github.io/НАЗВАНИЕ_РЕПОЗИТОРИЯ/`

## 🔑 Вход в админ-панель

- Перейдите по ссылке `login.html` или нажмите «Админ» в верхней планке.
- Пароль: **admin123**

## 📦 Управление данными

- Товары и заказы хранятся в localStorage вашего браузера.
- При первом запуске данные загружаются из `data.json`.
- Для переноса данных используйте кнопки «Экспорт» и «Импорт» в админ-панели.

## 🛠️ Возможности

- Каталог товаров с фильтрацией по типу и сортировкой.
- Корзина с выбором размера и плёнки.
- Оформление заказа с сохранением в localStorage.
- Админ-панель для управления товарами (CRUD).
- Управление заказами (изменение статуса, просмотр, удаление).
- Тёмная тема.
- Чат с заготовленными ответами.

## 📁 Структура файлов

- `index.html` – главная страница
- `product.html` – страница товара
- `admin.html` – админ-панель
- `orders.html` – управление заказами
- `login.html` – вход в админку
- `style.css` – основные стили
- `admin.css` – стили админ-панели
- `script.js` – основная логика
- `product.js` – страница товара
- `admin.js` – админ-панель
- `orders.js` – заказы
- `data.json` – начальные товары
- `orders.json` – начальные заказы (пустой)
- `images/` – папка с изображениями

## ⚠️ Важно

- Данные сохраняются только в вашем браузере. При очистке кэша или смене устройства они потеряются.
- Для постоянного хранения используйте экспорт/импорт или перейдите на серверную версию.
